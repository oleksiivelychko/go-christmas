# go-christmas

Run `go run main.go christmastree.go` to see the animated Christmas tree decorated by snowflakes.

![Christmas tree](screenshot.png)

...or try the executable binary from the [Releases](https://github.com/oleksiivelychko/go-christmas/releases) page.

P.S. Increase the column width in your terminal up to 2.0 in case of discrepancy.
